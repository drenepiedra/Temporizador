jpackage empaqueta un runtime (JRE) por defecto, así el usuario no necesita Java instalado.

Requiere JDK que incluya jpackage (OpenJDK11+ con herramientas o Oracle JDK).

Alternativa clásica: Launch4j para crear un .exe que lance el JAR; luego empacar en un instalador con Inno Setup o similar.

Distribución

Si usas JAR: distribuir TimerCircle.jar + instrucciones para ejecutar con javaw.
Si usas jpackage: distribuirás un instalador/EXE que instala la aplicación como cualquier otra app de Windows.
Sugerencias útiles

Usa javaw.exe en accesos directos para evitar consola visible.
Si quieres icono en la ventana y en el EXE, crea icono.ico y pásalo a jpackage o configúralo en Launch4j.
Para actualizar el programita fácilmente, crea un pequeño script build.ps1 que compile y cree el JAR automáticamente.
